# Vert.x S3 client

A fully functional Vert.x client for S3

## Compatibility
- Java 8+
- Vert.x 3.x.x

## Dependencies

### Dependency Vert.x 3.x.x
### Maven
```xml
<dependency>
    <groupId>com.hubrick.vertx</groupId>
    <artifactId>vertx-s3-client</artifactId>
    <version>1.0.0</version>
</dependency>
```
